# COMP2110 Week 10

## Define Web Application
1. What problem does the app solve? 
-  Study Planner Management App: This app allows for students to organise and manage their time with tracking tasks, organising study sessions, and stay motivated by breaking work into manageable chunks. 
- Students often struggle with procrastination and poor organisation. This app helps them plan study time and track progress in one place. 

2. Target Users: 
- University students
- High School students
- Anyone managing study/work tasks

3. Key Features: 
- create and delete tasks
- set deadlines for tasks 
- Mark tasks as complete 
- simple dashboard showing progress

## Analysis Web Application
1. Functional Requirements
- Users can create, edit, and delete tasks
- Users can mark tasks as completed
- The app displays a list of all tasks in a dashboard

2. Non-functional Requirements 
- The app must load within 2 seconds on standard internet connection
- The interface must be moblie-friendly and responsive.
